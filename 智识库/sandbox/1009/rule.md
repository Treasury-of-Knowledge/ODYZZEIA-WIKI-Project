# 🧭 **类脑RULE：社区基本准则**

为了确保每位成员都能在此享受一个 **健康、安全、富有创造力** 的交流环境，我们共同制定以下行为准则。  
请 **仔细阅读本准则并严格遵守**，这是维护我们共同家园 **和谐稳定的基石**。

# 🛑 **红线违规：零容忍区域**  
**触犯以下任何一条规则，都将导致 `立即、无条件的永久封禁`。**

## **1. 商业化行为（禁止任何形式的牟利）**

本社区为 **非营利性质** 的交流平台，**严禁利用社区资源进行商业牟利**。

**禁止范围：**  
- **利用社区资源牟利**：如利用 AIRP、其产品或服务获取利润或预期利润。  
- **接受赞助**：带有回报条件的。  
- **商业化广告**：任何直接或间接的产品宣传、诈骗信息、消费诱导。  
- **诈骗 / 诱导信息**：包括伪装成社区资源的诈骗内容。  
- **商业化引流**：将成员导向任何商业社群或平台。  
- **商业化推广**：如捆绑赠送、现金或等值奖励。

**适用范围包括但不限于：**  
- 前端工具（如 SillyTavern 等）  
- 预设 / 角色卡  
- LLM 语言模型相关内容

## **2. 危害社区及成员安全**

我们致力于保护每一位成员和社区整体的安全。

**🔒 针对个人成员的严重侵害：**  
- **公开隐私（“开盒”）**：违背意愿泄露他人真实信息。  
- **人身威胁**：有明确指向的威胁行为。  
- **恶意举报**：无差别针对他人进行大规模举报。
- **恶意代码**：在插件/脚本或独立前端等工具中插入恶意代码，对使用者的权益产生威胁。

**🛡️ 针对社区整体的攻击：**  
- **攻击与窃取**：如技术入侵、社工手段等非法获取内容。  
- **滥用
- **恶意引流（“破圈升堂”）**：将恶意流量引入社区，破坏环境。

## **3. 特定严重违规及规避处罚**

以下行为同样属于 **红线级别**，一经确认将 **严肃处理**：

- **发布涉及真实未成年人的高风险内容**：包括高度拟真的 AI 生成内容。  
- **使用小号规避封禁**：明确关联至已被封禁的账号。  
- **新成员的严重违规行为**：加入未满一个月即触犯严重违规条款。
- **违背作者意愿**：无视作者选择的协议或条款，擅自对作品进行二传或二创。

# ❌ **严重违规**

若出现以下行为，处理措施包括但不限于：**重新答题、整改、临时禁言，并会被赋予【被警告者】身份组（最长90天）**。  
视严重情况或在警告期内再次违规，将可能进行**永久封禁**。

## **1. 危害社区群体**

- **利用漏洞（钻空子）**：寻找或利用社区规则或系统的漏洞，试图谋求特权或规避处罚。  
- **针对性群体攻击言论（“地图炮”）**：对社区内的特定用户群体或社区本身进行无差别攻击。  
- **借作品影射攻击**：以发布作品的形式，含沙射影地诋毁、攻击特定用户或群体。
- **教唆他人违反规则**：试图教唆他人违反规则，或是捣乱阻挠管理组进行判断的行为。

## **2. 煽动仇恨对立**

- **绕过正常渠道，煽动矛盾**：不使用投诉 / 举报 / 屏蔽等合规手段，而发表公开指责、跑马圈地、挑动男女矛盾的言论。  
- **政治话术攻击**：使用键政话术对特定个人或群体进行明显的攻击 / 贬低。  
- **敏感政治话题**：对两岸政治问题、中国政治体制、高度敏感的中国政治人物相关的话题
- **煽动其他仇恨**：包括但不限于煽动社会、团体、种族等其他仇恨 / 对立。

## **3. 发布高风险多媒体内容**

- **高风险真人图片 / 视频**：在任何位置发布高危真人图片 / 视频，或侵犯他人隐私的内容（包括高度拟真的 AI 内容）。  
- **发布有害文件**：传播包含病毒 / 有攻击性 / 涉及犯罪的文件。

## **4. 不当求助行为**

- **求助违规渠道：**要求违规云服务或API渠道相关协助
- **过度要求作者**：频繁要求作者解决非作品本身导致或存在的问题。  
- **无礼伸手（“爷新行为”）**：不阅读规则教程，频繁无礼貌询问。

## **5. 不尊重原创 / 付费服务相关**

- **寻求有偿服务**：寻求付费咨询、代写、远程教学等有偿服务。  
- **侵犯作者权益**：违背作者意愿对作品进行的其他经核实行为。
- **公开讨论第三方**：在社区内任意频道或子区讨论第三方渠道。

# ⛔ **其他违规行为**

以下行为将根据情节严重性，面临 **警告、重新答题、整改、禁言、“小黑屋”** 等处罚。

## **1. 自我中心行为**

- 在深渊相关区外讨论 / 发表超常规性癖内容。  
- **人身攻击 / 骚扰 / 泄露隐私**，被当事人投诉。  
- **反串拱火、阴阳怪气、挑拨离间**。  
- 大量输出负面情绪、无理取闹、刷屏骂街。  
- 其他令他人明显不适的行为。

## **2. 越界创作**

- **资源区进行无作品的预告引流**（锁定）。  
- **影射指向现实人物的作品**（整改）。  
- **影射现实事件的作品**（删除）。  
- **高度争议的二创**（警告）。

## **3. 变相宣传广告**

- **公开寻求非官方 AI 渠道**。  
- **传播有收益的邀请链接**。

  # 📢 **社区劝导：避免讨论特定话题**

为了维护社区的**纯净与友好氛围**，我们劝导您**避免在公共频道讨论以下话题**：

## 🚫 **第三方交易平台**  
在 AIRP 相关事物上，**请避免推荐或宣传 淘〇、闲〇 等平台上的商品或商家**。

## 🚫 **真实赌博内容**  
请避免讨论现实或网络中的赌博、博彩、彩票等话题，  
例如：**线上博彩平台、彩票技巧、赌博经验分享等**。

## 🚫 **现实性服务**  
请避免涉及**金钱交易的现实世界性服务内容**的讨论，  
如：嫖〇、风俗等相关话题。

## 🚫 **引流与外宣行为**  
- 请避免在聊天区、公屏等**公开频道频繁引流至其他服务器**，此类行为可能打扰他人正常交流，也会增加管理负担。  
- 请避免在**公开社交平台对本社区进行大规模宣传**，以免带来高风险的曝光度。
- 请避免用开玩笑的方式**直接提供第三方的具体链接和联系方式**，有敏感信息请先对图片进行**打码处理**。在有不知情的社区成员认真询问的时候，也必须避免误导他人。

## ✅ **违反劝导的处理方式说明**  
- 我们鼓励大家使用**自助管理机器人**删除此类不当消息。  
- 若情况未改善，**管理员将介入提醒**；如不配合，将由管理员删除并**予以口头警告**。  
- 若多次警告无效，将视为违规行为：  
  - **添加“被警告者”身份组**  
  - **警告期内再次触发，将视为正式违规**  
  - 根据情节严重性**进行禁言等处理**

  # 📌 **重要声明**

## 🔞 **年龄限制**

加入本社区即表示该成员**自愿确认其已年满其所在地区法律规定的法定成年年龄**（通常为 18 周岁），  
具有人格独立能力与基本判断力，**成员不得主张任何形式的豁免**。

### 🕒 **投诉时效**
- 对于 **非红线** 且 **非持续发生** 的违规行为，投诉有效期为行为发生后 **7 天内**。
- 对于 **触犯红线** 或 **情节严重** 的行为，**无追溯期限制**。

### 🧾 **已处罚行为**
- 若某行为已被处理，**后续再有投诉将综合考量、酌情处理**，  
  但 **不会进行无休止的“翻旧账”**。

### 📅 **规则适用范围**
- 所有规则及后续更新**仅适用于 2024 年 9 月 27 日之后发生的行为**。  
- 在此之前的行为，若之后未再重犯，**则不予追溯**。

## 🪓 **针对商业化社区的额外条款**
 - **在任何商业化社区发生针对类脑社区以及成员的违反规则之行为，管理组保持处罚的权利**

# 📜 当前社区制度：有制约反馈的民选制度

## 1. 🧭 总管理  
- **人数**：目前 5 人  
- **产生方式**：由选举产生  
- **任期**：3 个月  
- **选举门槛**：  
  - 具备创作者或社区助力者身份组  
  - 满足社区活跃时长要求  
  - 无重大违规记录  
  - 情绪稳定
- **职责与权限**：  
  - 负责社区的日常运营、维护秩序、处理纠纷  
  - 制定和修改社区的基本规则（草案需公示并接受反馈）  
  - 根据判例进行裁量，若无判例则自由裁量提供判例，并公开决策逻辑  
  - 规划社区发展方向、落实具体计划、组织重要活动
  - 对紧急事务拥有最终裁决权（事后需公示理由）  
  - 审批社区建设者提出的方案并提供权限支持  

## 2. ⚙️ 执行管理
- **人数**：目前 8 人  
- **产生方式**：由选举产生  
- **任期**：3 个月  
- **选举门槛**：  
  - 拥有创作者或社区助力者身份组  
  - 满足社区活跃时长要求  
  - 无重大违规记录  
  - 情绪稳定
- **职责与权限**：  
  - 负责社区的日常运营、维护秩序、处理纠纷  
  - 根据判例与规则对违规行为进行处罚  
  - 协助社区发展，参与组织重要活动  
  - 负责具体事务的执行  

## 3. 📜 秘书组
- **人数**：根据职位不定
- **产生方式**：由管理组招募  
- **任期**：无任期限制（管理组可决定）
- **职责与权限**：  
  - 按照具体的负责事项区分  
  - 当前有赛事委员、提案委员、记者    

## 4. ⏳ 空缺期与临时管理  
- **空缺期**：管理组因失能或弹劾而空缺的时期，admin 将接管运营职责并启动临时管理  
- **临时管理**：  
  - **产生方式**：由退任管理优先组成；若人数不足，由 admin 招募补足  
  - **职责与权限**：  
    - 负责空缺期内社区的日常运营、维护秩序、处理纠纷  
    - 组织新管理招募（临时管理可参与竞选）  

## 5. 🏗️ 社区运营区：社区建设者  
- **门槛**：拥有创作者或社区助力者身份组  
- **职责与权限**：  
  - 可在社区提交运营或活动相关提案  
  - 提案通过初步审核后开启帖子供社区成员讨论  
  - 提案需制定执行方案，社区建设者达成一致或获得 66% 多数支持后提交管理组审核（假如影响较大需要扩大投票规模）
  - 若在时限内未提出合理方案，由管理组反馈并处理  

## 6. 🧾 反馈与制约  

### 🔨 弹劾机制（3种方式）  
1. **管理组内部弹劾**：管理组成员可发起弹劾投票，过半数即成立  
2. **成员投诉弹劾**：成员可基于原则性问题或重大失误发起弹劾，需创作者和社区助力者投票过半数通过  
3. **admin 弹劾**：admin 达成一致后可弹劾管理组成员，需给出理由并公示  

### 📣 投诉机制  
- 社区成员可发起私密或公开投诉  
- 投诉对象可包括管理组成员  
- 正当投诉将促使管理组自我纠正；若多次未改，admin 将介入处理  

## 7. 🔧 制度改革  

- 管理组与社区成员可总结制度运行中存在的问题  
- 需要与 admin 沟通改革方案  
- 经 admin 同意后可执行制度调整

## 8. 🧷 管理权限与社区运营事务说明

### ✅ 管理组可以做什么？
1. **社区运营的决策**  
   - 管理组有权开展活动或实施措施、自主的建设社区
   - 按既定规划推进落实，例如设立新类别与频道以及制定各频道的具体规则、针对非rule的规则进行公告、限制未被明确的危险行为并发布警示
2. **违规处罚的决定**  
   - 管理组可依据现有判例裁定违规处罚  
   - 若无相关判例，可实行自由裁量并创设新判例  

### ❌ 管理组不能做什么？
1. **直接变更制度与规则**  
   - 所有制度和社区rule需要严格经过 admin 监督
   - 管理组可提出建议修改，但需 admin 提出和修改 
2. **资源区的管理**  
   - 资源区的删除、重命名、结构调整等，非经 admin 确认不得进行
3. **删除非临时频道**  
   - 对于已确认并长期运行的频道，管理组不得擅自删除  
   - 若需调整，需先提交提案并获得通过  

## 9. 📢 公示试行期机制
- 管理组作出的社区运营决策需进行**公示**  
- 公示后可**立即进入试行阶段**，为期一星期  
- 社区成员可在试行期间对决策提出改革建议或投诉  
- 若无通过的改革建议或者投诉，试行期后可正式保留，否则可在试行期内或试行结束后改革或撤销该决策

# 📜 针对插件/脚本与独立前端等工具的补充规则说明

近期因插件相关问题引发多起事件，管理组现对各类插件脚本与独立前端作出以下补充规则（Rule）：

**1️⃣ 工具源代码公开要求**  
工具发布时虽不强制要求开源，但**必须公开完整源代码**(并非开源)，供使用者核查其中是否存在潜在安全风险。

**2️⃣ 信息收集与提示要求**  
若工具涉及信息收集等行为，作者须在显著位置**清晰告知用户**，并提供**关闭或拒绝该功能的选项**。

**3️⃣ 第三方请求披露要求**  
作者需在**帖子首楼或其他醒目位置**披露所有**不在白名单内的第三方请求域名**。

**⚠️ 违规处理**  
管理组将对未遵守上述补充规则的工具进行提醒；若在限期内未作出整改，**将锁帖处理**。  
情节严重者，**将视情况对当事人进行处罚**。
