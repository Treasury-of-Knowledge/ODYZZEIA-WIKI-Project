「随行终端 ERALINK」

あなたのそばにへ（中文：伴你左右）

ERALINK 是基于「多端合一/极限便携」理念开发的 Termux 终端脚本，旨在解决工具部署的繁琐问题。

它的诞生源于一个纯粹的愿望：将 SillyTavern、ClewdR、gcli2api 等强大工具的「部署-管理」流程化繁为简，通过可扩展的模块化框架，提供一键式可靠体验。

致谢：本项目基于 rezline 一键部署脚本分支开发，特别感谢原作者 rzline、gcli2api 作者 su-kaka 等老师的指导与支持！

❖ 核心特性 (Core Features)

  • ✦ 模块化架构 (Modular Architecture)
    告别臃肿单体脚本！每个核心功能（如 SillyTavern 管理）均为独立可插拔模块，新增功能或维护时更灵活。
  • ✦ 动态化菜单 (Dynamic Menu)
    主菜单随「已启用模块」自动生成，支持在设置中「显示/隐藏」特定模块，定制你的专属操作面板。
  • ✦ 统一化管理 (Unified Management)
    依赖安装、版本检查、服务启停、SSH 配置、卸载等操作，均收纳于可视化界面，无需记忆零散命令。
  • ✦ 轻量化体验 (Lightweight Experience)
    基于纯 Shell 构建，无需繁重运行时环境，以最轻盈的姿态提供稳定服务。

❖ 核心功能 (Key Functions)

  • 模块安装与更新：一键安装/更新 ClewdR、SillyTavern、gcli2api 及 ERALINK 框架本身
  • 服务状态监控：主菜单实时显示模块状态（[运行中] / [已停止]），系统状况一目了然
  • 版本智能检查：启动时异步对比「本地版本」与「最新版本」，支持缓存与手动强制刷新
  • 模块显示管理：在设置中自由选择需显示的模块，简化菜单界面
  • 安全卸载：完整卸载管理器，可选择性移除模块、依赖、配置文件或 ERALINK 自身
  • 调试模式：为开发者提供详细后台日志输出，便于问题定位
  • SSH 管理：一键安装/启停 OpenSSH 服务，并配置开机自启
  • 网络代理：支持全局代理的开启/关闭、自定义地址与重置
  • 开机自启：一键配置 ERALINK 脚本自身的开机自启动

❖ 安装指南 (Installation Guide)

答疑说明：作者目前在 Discord 社区（elysianhorizon） 活跃，如需帮助请前往 原帖下方咨询~

1. 事前准备

  1. 一部安卓手机（鸿蒙/苹果系统需参考其他适配教程）
  2. （大陆用户）开启手机流量，暂不使用 WiFi
  3. （大陆用户）准备并启动可访问外网的代理工具（确保连接稳定）
  4. 推荐浏览器：Microsoft Edge（兼容性更佳）

2. 安装 Termux

在安卓手机上下载并安装 Termux 官方版本：
Termux 官方下载（GitHub Releases）

3. 配置 Termux 权限（关键！）

需开启以下权限，避免后台运行异常：

  • 基础权限：允许「自启动」「关联启动」「后台弹出页面」「存储权限」「管理所有文件」
  • 电量设置：允许「后台高耗电」，禁止「暂停闲置应用的活动」

注意：若出现「酒馆无响应、后台无报错」，大概率是手机限制后台进程，可尝试「挂小窗运行」解决。

4. 部署一键启动脚本

打开 Termux，粘贴下方指令执行（必要时保持代理连接）。可先 点击获取手机端适配指令（Discord 链接）。

方式一：无代理（推荐）

apt update && apt install curl unzip git nodejs jq expect -y && pkg upgrade -y && curl -L -o install.sh.tmp -C - https://github.com/404nyaFound/eralink/releases/latest/download/install.sh && mv -f install.sh.tmp install.sh && chmod +x install.sh && ./install.sh || { echo "安装过程中出错"; rm -f install.sh.tmp; exit 1; }

方式二：带代理（适配网络受限场景）

apt update && apt install curl unzip git nodejs jq expect -y && pkg upgrade -y && curl -L -o install.sh.tmp -C - https://ghfast.top/https://github.com/404nyaFound/eralink/releases/latest/download/install.sh && mv -f install.sh.tmp install.sh && chmod +x install.sh && ./install.sh || { echo "安装过程中出错"; rm -f install.sh.tmp; exit 1; }

执行说明：
安装过程中需 根据提示输入 y 或回车；若报错，可尝试 升级 Termux 版本 或 切换 Termux 镜像源。
安装完成后会自动显示 GUI 界面，可在系统设置中开关脚本自启动。

5. 安装模块（ClewdR / SillyTavern / gcli2api）

  1. 默认仅显示「SillyTavern（酒馆）」及系统设置，如需安装其他模块：进入「系统设置 → 管理模块显示」，勾选需显示的模块（如 ClewdR、gcli2api）。
  2. 返回主菜单，输入对应模块的数字编号，选择「安装」。
  3. 安装过程中根据提示输入 y 或回车，完成后按任意键返回主菜单，即可查看模块运行状态。

6. 使用模块

  1. 在主菜单输入模块对应的数字编号，启动服务。
  2. 操作技巧：按 ESC 右侧的「菜单键」可打开侧边栏，选择「NEW SESSION」创建新进程（推荐：一个进程运行 ClewdR，一个进程运行酒馆）。
  3. 酒馆访问：启动后会自动调用浏览器打开，也可手动输入链接（默认：127.0.0.1:8000）。
